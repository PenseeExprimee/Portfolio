1. TRANSFORMER UNE WEBCAM EN CAMÉRA TRACKEUSE GRÂCE À L'IA

Je construis une caméra trackeuse 📸

Impossible d’être passé à côté l’année dernière : la DJI Osmo Pocket 3 a fait un carton sur les réseaux sociaux!
Elle a séduit autant les créateurs de contenu que les amateurs qui veulent capturer leurs meilleurs moments du quotidien.
Petite, discrète et offrant une super qualité d’image, elle a tout pour plaire. Mais ce qui la rend vraiment impressionnante, c’est son mode ActiveTrack : la caméra reconnaît un visage et le suit automatiquement, comme si elle avait son propre caméraman intégré!
C’est cette fonctionnalité qui m’a donné l’idée pour un projet que j’ai appelé “caméra trackeuse”.

2. CRÉER VOTRE PREMIÈRE APPLICATION MOBILE

Je construis une caméra trackeuse 📸

J’ai voulu apprendre à coder une application.
Pourquoi? Parce quand un développeur souhaite que son code puisse être facilement utilisé par d’autres, il faut emballer ce code dans un joli packaging. J'ai donc regardé un tutoriel de 37 heures, sur plusieurs semaines même.
Le problème avec les tutoriels, c’est qu’ils sont utiles seulement si vous les utilisez de la bonne façon. C’est à dire que vous vous servez des connaissances acquises pour créer votre propre projet. Dans mon cas, le tutoriel que j’ai suivi permet de créer une simple application de notes.
J'ai décidé de me baser sur ce tutoriel pour créer une application appelée Rappelle Moi :D.
